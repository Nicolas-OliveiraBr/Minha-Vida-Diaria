# Minha Vida Diária
Essa é uma aplicação web simples que dá liberdade aos usuários para escreverem o que quiserem, especialmente sobre a sua rotina ou, mais especificamente, o seu dia. As principais tecnologias utilizadas no desenvolvimento desse projeto são o Node.js, Express, Templates EJS e a comunicação é feita a partir do banco de dados Mongo (MongoDB).
